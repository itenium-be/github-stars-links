Github Stars
============

Add Github Stars shields to Github links.  
Activates directly on `activateDirectlyOn` array. For other websites, press `Control + Alt + G`

```javascript
// Defaults +/- to
const activateDirectlyOn = ['stackoverflow.com', 'google.com', 'github.com'];
```
